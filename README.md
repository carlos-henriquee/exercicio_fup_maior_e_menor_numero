# This is a coding exercise made in C assigned in college to practice conditional structures.
### The program takes the 3 numbers entered by the user, calculates the witch is the largest and the smallest number and displays the result on the screen.
## How to run it?
### First, clone this repository to your machine.
### If you are on Windows, then make sure you already have gcc installed. If you don't, follow this [tutorial](https://dev.to/gamegods3/how-to-install-gcc-in-windows-10-the-easier-way-422j)
Then, open cmd and navigate to your working directory:
```
cd *path/to/the/file/directory*
```
Now, run the code usign the gcc compiler:
```
gcc maior_menor.c -o "name you want for the executable"
```

### If you are on Linux, you probably already have gcc. If you don't, just run this commands on your terminal:
```
sudo apt update
sudo apt install build-essential
```
Go to the directory where the file .c is:

```
cd *path/to/the/file/directory*
```


And then, run the code:
```
gcc maior_menor.c -o "name you want for the executable"
```
